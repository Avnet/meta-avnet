Avnet Python Web Server Demo

Hello! This is a demonstration of a simple Python embedded webserver. Using Python, a webserver can be easily embedded into your Linux system. A webserver provides an easy method to control or monitor the embedded platform via an Internet browser.

Controlling the Embedded System

This example is intended to illustrate how the functionality of the embedded system can be controlled from the browser. Here, the LEDs on the board are tri-color (red/green/amber) and the user can switch them on or off or select the color by clicking on the selector dropbox and 'Set PS LED' or 'Set PL LED' buttons. The PL LED has two modes. It can either be set to 'Microphone LED' and display audio detected on the microphone or it can be set to 'User LED' and display the color controllerd by this webpage.
